# Video Tutorial Script: Basic Usage and First Workflows

**Duration:** 12-15 minutes
**Target Audience:** Users who have completed installation
**Prerequisites:** YouTube MCP Extended installed and configured

## Tutorial Overview

This tutorial walks users through their first workflows with YouTube MCP Extended, covering video listing, metadata optimization, and basic playlist management. Viewers will complete real tasks and see immediate results.

## Production Notes

### Screen Recording Requirements

1. **Clean YouTube Channel:** Use a test channel with varied content types
2. **Prepared Examples:** Have specific video IDs ready for consistent demonstration
3. **Before/After Comparisons:** Show clear metadata improvements
4. **Real-time Processing:** Capture actual batch operations completing

### Visual Enhancements

1. **Highlight Boxes:** Draw attention to important parts of responses
2. **Split Screen:** Show Claude interface alongside YouTube Studio
3. **Progress Animations:** Visualize batch operation progress
4. **Comparison Tables:** Before/after metadata side-by-side

### Interactive Elements

1. **Pause Points:** Allow viewers to follow along with their own content
2. **Copy-Paste Commands:** Provide exact commands in description
3. **Checkpoint Summaries:** Recap progress at each section
4. **Troubleshooting Callouts:** Address common issues during demonstration

### Accessibility Features

1. **Detailed Narration:** Describe all visual elements
2. **Clear Code Reading:** Slowly read all commands and responses
3. **Progress Indicators:** Verbal progress through the workflow
4. **Error Handling:** Show how to recover from common mistakes
