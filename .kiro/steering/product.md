# Product Overview

YouTube MCP Extended is an AI-powered YouTube management system that integrates with Claude Desktop via the Model Context Protocol (MCP). The product enables content creators to optimize their YouTube workflow through intelligent automation and AI-assisted decision making.

## Core Value Proposition

- **AI-Powered Metadata Optimization**: Generate SEO-optimized titles, descriptions, and tags using Claude's AI capabilities
- **Strategic Content Scheduling**: Intelligently schedule video releases to maximize audience engagement
- **Automated Playlist Management**: Organize content efficiently with AI-driven categorization
- **Data Protection**: Comprehensive backup and restore system for metadata safety
- **Secure Authentication**: OAuth 2.1 with PKCE for secure Google/YouTube API access

## Key Features

- Video metadata analysis and optimization suggestions
- Batch processing for large-scale operations
- Real-time progress tracking for long-running tasks
- Automated backup creation before any metadata changes
- Granular restore capabilities for data recovery
- Strategic scheduling algorithms (optimal, spread, custom)
- Playlist creation and management automation

## Target Users

Content creators, YouTube channel managers, and digital marketing professionals who need to efficiently manage large volumes of YouTube content while maintaining high-quality metadata and strategic publishing schedules.

## Integration Points

- **Claude Desktop**: Primary interface via MCP protocol
- **YouTube Data API v3**: Core YouTube functionality
- **Google OAuth 2.1**: Secure authentication
- **Local Storage**: Encrypted token and backup storage
